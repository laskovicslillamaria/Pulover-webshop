# Pulóver webshop
## Mire jó a pulóver webshop?
Egy online webshop, ahol különböző stílusú pulóvereket lehet megtekinteni és megvásárolni. A cél egy letisztult, felhasználóbarát felület biztosítása, ahol a felhasználók gyorsan megtalálhatják a számukra megfelelő terméket.

## Előfeltételek, követelmények:
#### Az webshop használatához Edge, Chrome, Firefox böngészők szükségesek
#### Regisztráció és bejelentkezés nélkül elérhető

## Fő funkciók:
#### Az oldal megtekintése - A weboldal megtekintését elérheti bárki.
#### Termékek böngészése - Itt a termékek keresését, az árának és az adatainak a megnézését értjük.
#### Kosár használata - A kosárba bele lehet tenni a kívánt terméket és onnan törölni is lehetséges.
#### Rendelés leadása - A kosár tartalma kiírja a végösszegét a rendelésnek.
